Welcome to the (new) Browserify CDN

This is a work in progress.

How it works:

  1. http POST a javascript file to /. For example: curl -X POST -d 'var traverse = require("traverse");' browserify.jit.su
  2. The Browserify CDN browserifies your posted index.js for you!
  3. The Browserify CDN downloads your packages from npm if necessary!
  4. The Browserify CDN uses couchdb to cache browserify bundles so you don't have to wait!
  5. The Browserify CDN will even uglify-js your code into a compact package!

BASICALLY IT'S PRETTY SWEET.
